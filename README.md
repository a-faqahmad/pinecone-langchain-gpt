## Getting Started

To run the frontend server, install the dependencies then run:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

To run backend:

```bash
cd src/backend
# then
npm install
# then
npm start
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
